Bienvenidos

A continuación, se presenta un sistema de detección de polaridad sobre aspectos en reseñas de restaurantes para el idioma español. El sistema clasifica las reseñas mediante técnimas de procesamiento de lenguaje natural por medio de redes neuronales y muestra el resultado según la opción que se elija.



Ingresar reseña

Esta opción permite realizar la calsificación de una sola reseña ingresada mediante un cuadro de texto. Da como resultado la polaridad y los aspectos que contenga.


Elegir



Subir archivo

Esta opción permite realizar la calsificación de varias reseñas contenidas en un archivo de texto o en una BD. A la salida de la clasificación el sistema permitirá ver mediante gráficas los resultados.


Elegir



Consultar

Esta opción permite realizar una consulta a los archivos de reseñas ya clasificados que se han guardado en la base de datos del sistema mediante su número de identificación.
